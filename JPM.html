Matplotlib in Jupyter Notebooks

I started out learning Jupyter Notebooks. Jupyter Notebooks is a development environment that allows you to write executable code, documentation, notes, etc in isolated cells. Being able to include documentation and notes alongside your code execution allows data scientists to explain their data visualizations alongside their visuals.
